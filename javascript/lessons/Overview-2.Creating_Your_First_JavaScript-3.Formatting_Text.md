## Formatting Text  

Just like in HTML, we can use HTML tags to format text in JavaScript.  
For example, we can output the text as a heading.  

>###### |js|
>\<html><br/>
&nbsp; \<head>\</head><br/>
&nbsp; \<body><br/>
&nbsp; &nbsp; &nbsp; \<script><br/>
&nbsp; &nbsp; &nbsp; &nbsp; &nbsp;  document.write("<h1>Hello World!</h1>");
&nbsp; &nbsp; &nbsp; \</script><br/>
&nbsp; \</body><br/>
\</html>
